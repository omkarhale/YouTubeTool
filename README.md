# YouTube Tool

YouTube Tool is a Spring Boot application that provides various utilities for interacting with YouTube videos, such as retrieving video details, generating SEO tags, and fetching thumbnails.

## Features

- **Video Details Retriever**: Fetches detailed information about a YouTube video, including title, description, tags, and thumbnail.
- **SEO Tag Generator**: Extracts SEO tags from YouTube videos based on a given video title.
- **Thumbnail Fetcher**: Retrieves the thumbnail of a YouTube video using its URL or ID.

## Technologies Used

- **Backend**: Java, Spring Boot
- **Frontend**: Thymeleaf, TailwindCSS
- **Build Tool**: Maven
- **API Integration**: YouTube Data API v3
- **Dependency Management**: Lombok, WebClient

## Prerequisites

- Java 17 or higher
- Maven 3.6 or higher
- YouTube Data API key
